# 🚀 Projects

Active work with a clear goal and end state. Each project gets its own folder.

> [!example] Project folder structure
> ```
> Projects/
>   My Project/
>     Overview.md   ← goal, status, links to notes
>     Tasks.md      ← checklist of what needs doing
>     Notes/        ← research and working notes
> ```

## When something is a project

- It has a goal (a specific outcome)
- It has tasks (steps to get there)
- It will eventually be **done** (then move to [[Archive/README|Archive]])
