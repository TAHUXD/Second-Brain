# 📥 Inbox

Drop anything here without overthinking it — ideas, links, voice memo transcripts, random thoughts.

> [!warning] Process weekly
> The inbox should stay close to empty. Once a week, move notes to [[Notes/README|Notes]], [[Projects/README|Projects]], or [[Resources/README|Resources]].

## How to capture

- Quick idea → just make a new note here, title it anything
- Article to read → paste the link, Claude can scrape it into clean markdown later
- Meeting notes → dump them here first, clean up later
