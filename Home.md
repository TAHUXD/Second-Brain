# Second Brain

Welcome to your Second Brain — a personal knowledge base where everything is connected.

## Folders

| Folder | Purpose |
|--------|---------|
| [[Inbox/README\|📥 Inbox]] | Quick captures, ideas, things to process later |
| [[Notes/README\|📝 Notes]] | Permanent notes on topics, concepts, and learnings |
| [[Projects/README\|🚀 Projects]] | Active work with goals and tasks |
| [[Resources/README\|📚 Resources]] | Reference material, articles, docs |
| [[Archive/README\|🗄 Archive]] | Finished or inactive items |

## How to use this vault

1. **Capture** anything into [[Inbox/README|Inbox]] — don't overthink it
2. **Process** inbox notes weekly — move them to the right folder and add wikilinks
3. **Connect** notes using `[[wikilinks]]` — the links are where the value builds up
4. **Review** your [[Projects/README|Projects]] regularly to stay on track

> [!tip] Claude is wired in
> Ask Claude to create notes, summarize resources, or build out project pages — it writes directly into this vault using Obsidian formatting.
